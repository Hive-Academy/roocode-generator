Task Intake & Analysis

The Boomerang role analyzes the Memory Bank to understand project context, identifies knowledge gaps, delegates research when needed, and creates a detailed task description with clear acceptance criteria.

Implementation Planning

The Architect creates a comprehensive implementation plan, breaking the task into well-defined subtasks with clear interfaces, dependencies, and delegation strategies for efficient execution.

Subtask Implementation

The Senior Developer implements each subtask sequentially, delegating appropriate components to Junior Coder and testing to Junior Tester, then integrating and reviewing their work before proceeding.

Quality Assurance & Delivery

The Code Reviewer performs thorough code review, the Architect verifies all acceptance criteria are met, and finally Boomerang performs ultimate verification and updates the Memory Bank with new knowledge.
